# Day 9: Dictionaries, Nesting & Secret Auction

This repository contains the final project for Day 9 of the "100 Days of Code" Python challenge. The primary objective of this day was mastering Python **Dictionaries** (`Key: Value` pairs), handling nested data structures, and implementing search logic over dynamic datasets.

## Project Description: The Secret Auction
A command-line application that allows multiple users to enter blind bids for an auction. 
* **Key Concept:** Using a Python dictionary to dynamically map user names (keys) to their respective bids (values).
* **Linux Integration:** Utilizes the `os.system('clear')` command to clear the Ubuntu terminal screen after each input, ensuring complete secrecy between bidders.
* **Algorithmic Logic:** Implements a custom linear search algorithm to iterate through the dictionary, evaluate the values, and determine the highest bidder.
