A command-line tool that generates secure, randomized passwords based on user-defined criteria, implementing Python loops and the `random` module.

### Concepts Covered:
- **`for` Loops:** Iterating over Python lists and sequences efficiently.
- **`range()` Function:** Using `range(start, end)` to control loop execution counts.
- **Accumulator Pattern:** Dynamically building strings and lists inside a loop body (`password += char`).
- **Advanced `random` Methods:** Using `random.choice()` to select single elements and `random.shuffle()` to randomize list sequences in-place.
