# Day 18: Turtle Graphics & The Hirst Painting 

This repository contains the Day 18 project of the "100 Days of Code" Python challenge. The goal was moving from textual console environments into **2D GUI space coordinate manipulation** using coordinate geometry.

## 🛠 Project Description: Dynamic Art Generator
A script using turtle graphics to generate dotted matrix art modeled after Damien Hirst’s spot paintings. It calculates specific grid-offsets and loops state boundaries to map structured matrices.

* **RGB Color Mapping:** Integrates dynamic multi-value tuple parsing within graphics controllers.
* **Coordinate Grid Traversal:** Utilizes modulus arithmetic (`%`) and floor division (`//`) as structural anchors to calculate precise 2D spatial stepping for vector navigation.
