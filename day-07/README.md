# Day 7: Hangman Game

A classic text-based console game built as part of the "100 Days of Code" challenge. This project focuses on integrating fundamental programming concepts into a cohesive, working application.

## Core Concepts Practiced

* **Control Flow:** Combing `while` loops for the core game loop and `for` loops for string iteration.
* **Data Structures:** Managing lists to store the word bank, the current state of the hidden word (`_ _ _`), and tracked guesses.
* **Conditional Logic:** Implementing complex `if/elif/else` statements to validate user input and prevent duplicate guesses.
* **String Manipulation:** Working with string methods and indexing to dynamically update the game board.

## How to Run

Make sure you are in the project directory inside your terminal, the run:

```bash
python3 main.py
