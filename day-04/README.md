# Day 4 - Rock, Paper, Scissors Games

A classic command-line game that utilizes pseudo-random number generation, data validation, and Python lists for efficient data management.

### Concepts Covered:
- **Python `random` Module:** Generating random integers using `random.randint()`.
- **Data Structures (Lists):** Storing, organizing, and accessing elements inside a Python list via tracking indices.
- **Index-Based Mapping:** Using a user's integer input to dynamically retrieve ASCII art directly from a list (`game_images[user_choice]`).
- **Defensive Programming:** Implementing boundary checks to reject invalid user inputs before running the main logic.

```bash
python3 day-04/main.py
