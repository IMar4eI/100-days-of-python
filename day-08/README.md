# Day 8: Function Parameters & Caesar Cipher

This repository contains the projects completed during Day 8 of the "100 Days of Code" Python challenge. The focus of today was mastering functions with multiple inputs, positional vs. keyword arguments, and basic cryptographic logic.

## Projects Included

### 2. Caesar Cipher (Step 1)
A command-line tool that implements the classic Caesar Cipher technique to encrypt text messages by shifting letters forward in the alphabet.
* **Key Concept:** Advanced list indexing with `.index()` and preventing `IndexError` bugs when shifting characters near the end of the alphabet (`z`, `y`, `x`).
